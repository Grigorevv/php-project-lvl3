### Requirements

  * PHP ^7.4.0
  * Composer
  * SQLite for local
  
### Setup

  ```sh
  git clone https://github.com/Grigorevv/php-project-lvl3.git
  ```

  ```sh
  cd php-project-lvl3
  ```

  ```sh
  make setup
  ```
### Run

  ```sh
  make start
  ```
